# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in tabula, please report it privately
using [GitHub Security Advisories](https://github.com/rhalbersma/tabula/security/advisories/new)
rather than opening a public issue.

You should expect an initial response within a few days. Once a fix is
available, a new release will be published and the advisory disclosed.
